# Connecting to Chat
Скрипт подключается к чату и показывает переписку с участниками чата. Чат можно не только читать, но отправлять туда 
сообщения.

## Для работы
Скачать скрипт
```commandline
git clone https://github.com/Aleksashka301/connecting_to_chat
```
Установка виртуального окружения
```python
python -m venv myvenv
```
Активация виртуального окружения
```commandline
myvenv\Scripts\activate
```
Установка зависимостей
```python
pip install -r requirements.txt
```
### Переменные окружения
В корне проекта нужно создать файл `.env`, затем в этом файле создать переменные
 - `HOST` - домен, где находится сервер с чатом. Обязательная переменная, нужно заполнить или указать как аргумент при 
запуске скрипта. Флаг `--host`
 - `SERVER_PORT` - порт через который сервер проходит авторизация, регистрация и отправка сообщений. Обязательная 
переменная, нужно заполнить или указать как аргумент при запуске скрипта. Флаг `--server_port`
 - `CLIENT_PORT` - порт для чтения чата. Обязательная переменная, нужно заполнить или указать как аргумент при запуске 
скрипта. Флаг `--client_port`
 - `HISTORY` - название файла в который будет сохраняться история чата по умолчанию `history.txt`. Не обязательная 
переменная. Флаг `--history`
 - `TOKEN` - токен для авторизации в чате, если имеется. Если нет, чат предложит пройти регистрацию. Не обязательная 
переменная. Флаг `--token`

Запуск скрипта
```python
python main.py
```
